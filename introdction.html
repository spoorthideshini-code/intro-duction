CSS Concepts

CSS controls colors, spacing, and layout. Below are easy examples that show how styles can change the look and structure of a page.

Colors

Use color to style text and background-color to change backgrounds.

Layout

Use display, margin, and padding to arrange and space elements.

Tip: changing CSS does not require editing HTML structure. You can try new colors, fonts, or layout values in the stylesheet to update the whole page.

``` # Example CSS (style.css) ```css body { background-color: #f4f4f4; font-family: Arial, sans-serif; } .container { width: 70%; margin: auto; background: white; padding: 20px; border-radius: 10px; } .main-title { text-align: center; color: darkblue; } .section { margin-top: 20px; } h2 { color: purple; } p, li { font-size: 18px; color: #333; } ul { list-style-type: square; } hr { border: 1px solid gray; } ``` You can now easily change colors, fonts, spacing, backgrounds, borders, and more using CSS.
